자바 변수란
===  
자바에서 변수란 데이터의 저장과 참조를 위해 할당된 메모리 공간이다.  

데이터 타입
===

데이터 타입에는 정수와 실수,문자,참과 거짓이 있다 정수에는 byte,short,int,long 실수에는 float,double 문자에는 char 참과 거짓에는 bollean이 있다.

다음 예와 같은 a, b를 변수(variable)라고 한다.
~~~
int a;
String b;
~~~

if조건문
===
if는 조건이 맞으면 실행되는 영역이다
~~~
if (조건문) {
    <수행할 문장1>
    <수행할 문장2>
    ...
} else {
    <수행할 문장A>
    <수행할 문장B>
    ...
}
~~~
